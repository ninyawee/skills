# Recipe Note Creator

Turn recipe videos, URLs, or descriptions into visual step-by-step recipe notes saved to `~/life/`.

## Workflow

### 1. Extract recipe data

Depending on the source:

- **Video URL** — Use the `video-extract` skill to get metadata + transcript, then extract recipe info
- **URL (blog/article)** — Scrape and extract recipe
- **User description** — Use as-is
- **Existing ~/life/ note** — Read and enhance

Extract: dish name, servings, ingredients (with quantities), steps, tips, source URL.

### 2. Structure the recipe note

Save to `~/life/<dish-name>.md` with this format:

```yaml
---
tags: [research, recipe, <cuisine>]
source: <original URL if any>
author: claude
last_updated: YYYY-MM-DD
---
```

Body structure:
- `# <Dish Name>` (Thai name if applicable)
- One-line description
- `## Ingredients` — grouped by component (e.g., base, sauce, filling)
- `## Steps` — numbered, concise, one action per step
- `## Tips` — tricks and variations
- `#todo ลองทำ` at the bottom (so it shows up as a to-do)

### 3. Extract step images from video

Use `scripts/recipe-images.py` to extract real frames from the source video, then smart-crop to the subject using Gemini vision.

```bash
# Auto-detect 6 evenly-spaced key frames
scripts/recipe-images.py "<VIDEO_URL>" --auto --output-dir /tmp/recipe-<dish-name>/

# Specify exact timestamps (in seconds)
scripts/recipe-images.py "<VIDEO_URL>" --timestamps 15 30 45 60 75 --output-dir /tmp/recipe-<dish-name>/

# Skip smart crop (just center-crop to 4:3)
scripts/recipe-images.py "<VIDEO_URL>" --auto --no-crop --output-dir /tmp/recipe-<dish-name>/
```

The script:
1. Downloads the video via yt-dlp (lowest quality — only need frames)
2. Extracts frames at specified timestamps via ffmpeg
3. Uses Gemini vision to identify the subject (food, hands, pot) and smart-crop
4. Saves `thumbnail.jpg` (at ~33% for finished dish shot) + `step-N.jpg`

### 4. Choosing timestamps

If the video has chapters or the transcript reveals step transitions, pass specific `--timestamps`. Otherwise `--auto` distributes frames evenly across the video, skipping intro/outro.

For best results, watch the video description for time markers (e.g., "0:15 mix batter") and pass those as `--timestamps`.

### 5. Embed in note

After extracting, embed in the recipe note:

```markdown
![thumbnail](/tmp/recipe-<dish>/thumbnail.jpg)

## Steps

### 1. ผสมแป้ง (Mix batter)
![step-1](/tmp/recipe-<dish>/step-1.jpg)
...
```

Copy images into vault: `~/life/photos/<dish-name>/`. Reference with vault-relative paths (e.g., `photos/luem-kluen-mango/step-1.jpg`).

## Environment

- Requires `yt-dlp` and `ffmpeg` installed
- Smart crop requires `GEMINI_API_KEY` or `GOOGLE_API_KEY` (falls back to center crop without it)
- Uses `gemini-2.5-flash` for vision-based crop detection
